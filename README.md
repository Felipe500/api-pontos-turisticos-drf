# API DE PONTOS TURíSTICOS
Uma API que permitirá ver e cadastrar novos pontos turisticos juntamente 
com sua atrações.

Possibilidades:

- Deixar uma avalição do ponto turístico
- Deixar um comentário sobre o ponto turístico
